# 本地题库答题助手 — 使用说明

> **当前版本**：v1.5.0  
> **更新日期**：2026-07-17

## 安装

1. 打开 Chrome/Edge 浏览器，地址栏输入 `chrome://extensions/` 回车
2. 右上角打开「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `local-exam-helper` 文件夹
5. 浏览器右上角出现插件图标 ✅

## 考试操作流程

### 正确流程（不需要刷新页面）

```
1. 打开考试页面
2. 等题目全部加载完毕（约2-3秒，等 Element UI 渲染完成）
3. Ctrl+Shift+E → 右下角浮窗弹出
4. 鼠标悬停题目 → 浮窗显示匹配答案 → 自动勾选
5. 监考走近 → Ctrl+Shift+H 切隐形模式（浮窗消失，纯后台答题）
6. 监考离开 → Ctrl+Shift+E 恢复浮窗
```

> **注意**：不需要刷新页面。content script 在 `document_end` 自动注入，页面 DOM 构建完成后即可使用。如果按快捷键后浮窗没出来，大概率是题目还在加载，等2-3秒再按。

## 考前准备（提前做，避免考试时触发失焦）

### 导入题库

1. 点击插件图标 →「题库管理」
2. 如果有 Excel 题库：直接拖入或点「导入题库」选择文件
3. 如果有 JSON 题库：同上
4. 导入后勾选本次考试需要的题库（可多选）
5. 拖拽调整题库优先级（排前面的优先匹配）
6. 关闭管理面板

### 题库格式

**Excel（推荐）**：
| 题型 | 题干 | 选项A | 选项B | 选项C | 选项D | 答案 | 解析 |
|------|------|-------|-------|-------|-------|------|------|
| 单选 | 1+1等于几 | 1 | 2 | 3 | 4 | B | |
| 多选 | 哪些是颜色 | 红色 | 蓝色 | 桌子 | 绿色 | ABD | |
| 判断 | 地球是圆的 | | | | | 正确 | |
| 填空 | 中国的首都是 | | | | | 北京 | |

题型填：单选 / 多选 / 判断 / 填空  
多选答案连写字母（如 ABD），判断填 正确/错误

**JSON**：见 `question_template.json`

## 考试中使用

### ⚠️ 考试期间严禁点击插件图标

点击插件图标 → popup弹出 → 页面失焦 → **可能触发切屏检测**。

**安全做法**：
- **考前**：点图标打开popup → 管理题库、调整设置 → 关掉
- **考试中**：只用 `Ctrl+Shift+E` / `Ctrl+Shift+H`，绝对不碰插件图标

### 开启
按 `Ctrl+Shift+E`（考试中唯一的安全操作方式）

### 自动答题
打开后插件自动识别页面题目并勾选答案。右下角出现悬浮窗。

**中途开启自动纠错**：考试做到一半再开 → 已做对的跳过 → 做错的自动纠正 → 空着的补上。

### 悬浮窗说明
- **绿色高亮** = 高置信度匹配，已自动勾选
- **黄色** = 低置信度，需人工确认
- **红色「答案存疑」** = 多题库冲突/匹配度不足，**未自动勾选**，请人工判断
- 鼠标悬停题目 → 悬浮窗切换显示该题详情
- 可拖拽移动、右下角拖拽缩放

### 关闭
再次按 `Ctrl+Shift+E`，所有注入痕迹消失，页面恢复原样

## 快捷操作

| 快捷键 | 功能 | 界面 |
|--------|------|:---:|
| `Ctrl+Shift+E` | 普通模式开关 | 悬浮窗可见 |
| `Ctrl+Shift+H` | 隐形模式开关 | **零界面，纯后台** |

- **普通模式**：右下角悬浮窗 → 鼠标悬停题目看详情 → 自动纠错
- **隐形模式**：监考走动时使用，不显示任何界面，纯后台自动答题
- 两种模式可以随时切换

## 反检测能力

### v1.5.0 新增：DOM 签名随机化

之前版本所有注入元素使用固定 `__leh_` 前缀（14个ID/class），考试页面一行代码即可检测。现已改为每次注入时生成随机前缀：

```
之前: document.querySelector('[id*="__leh_"]')  → ✅ 抓到
现在: document.querySelector('[id*="x7a3b9c2d1e"]')  → ❌ 前缀每次不同，无法固定检测
```

CSS 样式也改为动态注入 `<style>` 标签，不再走静态 `content.css` 文件，避免页面 JS 遍历 `document.styleSheets` 发现非本站规则。

### 实战判断

国网考试系统的反作弊重心在内网隔离 + VPN + 摄像头监考 + 考试客户端锁定浏览器，页面内嵌 JS 反作弊几乎没有先例。使用 Supermium 便携版本身已绕开客户端锁定（客户端检测的是标准 Chrome 注册表路径）。当前版本直接上考场，风险极低。

## 常见问题

**Q: 切屏会被检测吗？**  
A: 不会。所有操作在页面内完成，不切换窗口、不失焦。

**Q: 选项顺序打乱影响吗？**  
A: 不影响。匹配基于答案文本，与ABCD序号无关。

**Q: 多题库答案不一致怎么办？**  
A: 该题不会自动勾选，悬浮窗显示多组结果及来源，人工判断。

**Q: 识别不出题目？**  
A: 检查页面是否是标准表单（radio/checkbox）。如果是Canvas渲染或自定义组件，当前版本不支持。
