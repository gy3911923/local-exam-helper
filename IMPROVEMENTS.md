# 改进建议与已知局限

## 已采纳的改进（v1 包含）

| 改进点 | 原PRD | 修正后 |
|--------|-------|--------|
| 扩展隐身 | 无 | manifest随机名 + 动态注入 + 无全局变量污染 |
| 题库存储位置 | content script IndexedDB | background IndexedDB，跨域名通用 |
| 模式分层 | 单一自动模式 | 普通模式 + 手动辅助模式 |
| 题库预置 | 仅导入 | 支持打包预置 + 在线导入双轨 |

## v2 计划迭代项

| 功能 | 优先级 | 说明 |
|------|:---:|------|
| Canvas截图OCR兜底 | 🔴 | 考试系统防作弊升级时的生存保障 |
| 自然模式（模拟鼠标轨迹+思考停顿） | 🔴 | 应对行为分析检测 |
| 答题进度统计 | 🟡 | 悬浮窗显示已答/总数/成功率 |
| 题库分组管理 | 🟡 | 按考试类型分组，一键切换 |
| 快捷键自定义 | 🟢 | 允许用户修改快捷键 |
| 错题标记与反馈 | 🟢 | 手动标记错误匹配，优化后续 |

## 已知局限（需用户认知）

1. **非表单类考试不支持**：Canvas渲染、Flash/WebGL渲染、纯图片题目无法处理
2. **跨域iframe受限**：部分考试系统的iframe跨域时content script虽能注入但DOM访问受同源策略限制
3. **移动端不支持**：Chrome Android不完全支持扩展，仅桌面端可用
4. **题库需自行整理**：不提供预置题库，用户需自行导入
5. **纯文本匹配**：不支持图片题、音频题、视频题
