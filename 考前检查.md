# 考前检查清单 v1.6.3

## 考前准备（外网）

- [ ] 1. 扩展名显示为「页面辅助工具」（隐身化）
- [ ] 2. `chrome://extensions/shortcuts` 确认快捷键已绑定
- [ ] 3. **`chrome://settings/downloads` → 位置改为「桌面」**（否则保存的文件在下载目录找不到！）
- [ ] 4. 题库管理 → 导入 Excel 题库 → 导出全部备份 JSON
- [ ] 5. 打开 test_exam.html → Ctrl+Shift+E → 题目全部匹配成功
- [ ] 6. Ctrl+Shift+S → 桌面出现 exam_page_xxx.html + _debug.json → **确认保存成功**
- [ ] 7. 拷贝整个插件文件夹 + 备份 JSON 到内网

## 考试当天（内网）

- [ ] 1. 加载扩展 → 绑定快捷键
- [ ] 2. `chrome://settings/downloads` → 位置改为「桌面」
- [ ] 3. 题库管理 → 一键恢复备份 JSON → 勾选题库激活
- [ ] 4. 关掉题库管理页面
- [ ] 5. 进入考试页面 → 等题目加载完 → Ctrl+Shift+E
- [ ] 6. 正常模式答题，监考走近按 Ctrl+Shift+H 切隐形
- [ ] 7. **考试结束前按 Ctrl+Shift+S 保存页面**（双文件落桌面）
- [ ] 8. 考后把 .html + _debug.json 带回外网做分析

## 快捷键总表

| 按键 | 功能 |
|------|------|
| Ctrl+Shift+E | 普通模式（浮窗 + 悬停答题） |
| Ctrl+Shift+H | 隐形模式（纯后台答题，零界面） |
| Ctrl+Shift+S | **保存双文件到桌面**（.html + _debug.json） |

## 保存后你会看到什么

按 Ctrl+Shift+S 后浮窗显示：
```
💾 已保存到「下载」目录

  exam_page_20260720_114305.html
  exam_page_20260720_114305_debug.json

⚠ 考前务必把 chrome://settings/downloads
     位置改为「桌面」，否则找不到文件！
```
如果已设好下载位置为桌面 → 直接去桌面找这两个文件。
